# SearchingWordCount
javaFX를 이용한 단어 빈도수 계산기

# 기본 사항
 * Java 1.8 이상
 * JavaFX

# 실행 방법
<pre>
java -jar SearchingWordCount.jar
</pre>

### Word Sample
~~~ bash
A
B
~~~

### Contents Sample
~~~ bash
AABCCCC
DDDDFFFGGGHHH
~~~

### output File Sample

~~~ bash
[A=2][B=1]|#1|AABCCCC

A=2,B=1
TOTAL COUNT=3
~~~
